# Hejia Cloud Community Management Platform

The **Hejia Cloud Community Management Platform** is a modern *Smart Property Management Solution*, designed as a professional **Internet+ community property management system**.

The platform integrates multiple functional modules such as **community asset management, residential management, visitor management, online maintenance requests, complaints handling, and community interaction**, aiming to significantly improve the efficiency of property staff and the overall quality of community services.

Its core mission is to build a **resident-oriented, efficient, and intelligent community service platform**.



---

## Core Modules

### 1. System Management
- User and role permission management  
- Department and position management  
- Dictionary data management  
- System log management  

### 2. System Monitoring
- Online user monitoring  
- Server status monitoring  
- JVM runtime and performance metrics  

### 3. Community Asset Management
- Community information management  
- Building management  
- Unit management  
- Housing asset management  

### 4. Residential Management
- Property owner information management  
- Owner verification and approval  

### 5. Community Interaction
- Community forum content management  
- Resident interaction and communication

## Technology Stack

### Frontend

| Technology | Description |
|----|----|
| Vue CLI | Scaffolding tool for building Vue.js applications |
| Element UI | Vue-based UI component library for admin systems |
| Node.js | JavaScript runtime environment |
| Axios | Promise-based HTTP client for API communication |

### Backend

| Technology | Description |
|----|----|
| Spring Boot | Rapid application development framework for Spring |
| Spring Security | Security framework for authentication and authorization |
| MyBatis-Plus | Enhanced ORM tool based on MyBatis |
| Redis | In-memory data store for caching and performance optimization |

- Promote digital and intelligent community services

# 合家云社区物业管理平台（Hejia Cloud Community）

合家云社区物业管理平台是一个全新的 **“智慧物业解决方案”**，是一款基于 *互联网+* 理念的专业社区物业管理系统。

平台通过 **社区资产管理、小区管理、访客管理、在线报修、意见投诉、社区互动** 等多个功能模块，全面提升社区工作人员的工作效率与物业服务质量，致力于构建一个 **以居民为中心、服务高效、管理智能** 的现代化社区服务平台。



---

## 系统主要模块

### 1. 系统管理模块
- 用户与角色权限管理  
- 部门与岗位管理  
- 字典数据管理  
- 系统日志管理  

### 2. 系统监控模块
- 当前在线用户信息  
- 服务器运行状态监控  
- JVM 运行参数与性能信息  

### 3. 社区资产管理模块
- 小区信息管理  
- 楼栋信息管理  
- 单元信息管理  
- 房屋资产信息管理  

### 4. 小区管理模块
- 业主信息管理  
- 业主审核与认证  

### 5. 互动信息模块
- 社区论坛信息管理  
- 居民互动内容发布与维护
